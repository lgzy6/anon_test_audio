### 2.2 Target Pool 构建问题

**当前 `pool_building.py` 的问题：**

| 问题 | 描述 | 影响 |
|------|------|------|
| **符号分配时机** | 在 pool 构建时分配 symbols | 符号与特征绑定，但检索时用错误的 z_smooth 匹配 |
| **无 Pattern 聚类** | Pool 未按说话人风格/Pattern 分组 | 无法实现"选择不同风格的目标池" |
| **Phone Clusters 混合** | 同一音素的所有说话人特征混在一起 | 失去说话人多样性控制 |

**`pool_building.py:267-312` 问题代码：**

```python

    symbols = dist.argmin(dim=-1).numpy()  # 正确的符号
    frame_to_symbol[start:end] = symbols
    # 保存到 symbols.npy
```

然后在 `retriever.py:234-238`：

```python
# 问题: 用被污染的 z_smooth 去匹配 Pool 中正确的 symbols
```

### 4.2 Pattern 聚类策略

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          Pattern 聚类方法                                    │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  输入: 所有说话人的特征序列                                                   │
│                                                                             │
│  Step 1: 提取说话人级别的 Pattern 表示                                       │
│  ┌────────────────────────────────────────────────────────────────────┐    │
│  │  对每个说话人:                                                       │    │
│  │    1. 收集该说话人所有帧的 SAMM 符号: z_spk [T_spk]                   │    │
│  │    2. 计算符号分布直方图: hist [K] = count(z == k) / T_spk           │    │
│  │    3. 计算转移矩阵: trans [K, K] = P(z_t | z_{t-1})                  │    │
│  │    4. 拼接为说话人表示: repr_spk = [hist, flatten(trans)]           │    │
│  └────────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│  Step 2: 聚类说话人                                                         │
│  ┌────────────────────────────────────────────────────────────────────┐    │
│  │  repr_all = stack([repr_spk1, repr_spk2, ...])  # [S, K + K*K]      │    │
│  │  kmeans = KMeans(n_clusters=n_patterns)                             │    │
│  │  pattern_labels = kmeans.fit_predict(repr_all)  # [S]               │    │
│  │  pattern_centroids = kmeans.cluster_centers_    # [n_patterns, D]   │    │
│  └────────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│  Step 3: 分配帧到 Pattern 池                                                │
│  ┌────────────────────────────────────────────────────────────────────┐    │
│  │  for spk_idx, pattern_id in enumerate(pattern_labels):              │    │
│  │      # 该说话人的所有帧归入对应 Pattern 池                            │    │
│  │      pool_pattern_{pattern_id}.append(features[spk_idx])            │    │
│  └────────────────────────────────────────────────────────────────────┘    │
│                                                                             │
│  输出:                                                                       │
│  - pattern_centroids.pt: 用于推理时预测源说话人的 Pattern                    │
│  - pool_pattern_{0..n}/: 按 Pattern 分区的特征池                            │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
